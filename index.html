O que é uma Publicação Web?

A publicação web é o processo de disponibilizar arquivos, páginas ou aplicativos na internet. Para isso, é necessário um servidor web que hospede os arquivos e os torne acessíveis a partir de um domínio.

Tipos de Aplicações

  • Aplicações Estáticas: Compostas por arquivos fixos (HTML, CSS, JS) que não mudam sem intervenção manual.
  • Aplicações Dinâmicas: Geram conteúdo em tempo real, conectadas a bancos de dados e utilizando linguagens como PHP ou Node.js.

O que é um Domínio e um DNS?

Um domínio é o nome amigável usado para acessar um site (exemplo.com). O DNS (Sistema de Nomes de Domínio) é o sistema que traduz esse nome para o endereço IP do servidor que hospeda o site.

Servidores: Compartilhado vs Dedicado

Servidor Compartilhado

Vários sites compartilham os mesmos recursos do servidor. É mais barato, mas oferece menor desempenho.

Servidor Dedicado

Um servidor exclusivo para um único cliente. Oferece maior desempenho, mas é mais caro.

O que é um Servidor Web?

Um servidor web processa e entrega páginas da web para os usuários. Exemplos:

  • Apache: Open-source, amplamente usado.
  • Nginx: Mais rápido, ideal para alta carga.

Backup e Restauração de Banco de Dados MySQL

Utilizando o arquivo banco_tarefas.sql, vamos demonstrar como realizar o backup e a restauração de um banco de dados MySQL.

1. Realizar o Backup de um Banco de Dados MySQL

O backup cria uma cópia dos dados em um arquivo .sql.

Passo a Passo:

  1. Abra o terminal ou prompt de comando.
  2. Use o comando mysqldump:
mysqldump -u [usuário] -p banco_tarefas > backup_banco_tarefas.sql
  • [usuário]: O nome do usuário do banco de dados.
  • banco_tarefas: Nome do banco de dados a ser copiado.
  • backup_banco_tarefas.sql: Nome do arquivo onde o backup será armazenado.

2. Restaurar o Backup de um Banco de Dados MySQL

Restaura o banco de dados a partir do arquivo .sql.

Passo a Passo:

  1. Certifique-se de que o banco de dados existe. Se não, crie-o:
CREATE DATABASE banco_tarefas;
  1. Use o comando mysql para restaurar:
mysql -u [usuário] -p banco_tarefas < backup_banco_tarefas.sql

3. Processo de Backup e Restauração (Ilustração)

Explicação Visual:

  • Backup: O comando mysqldump exporta os dados do banco banco_tarefas para o arquivo backup_banco_tarefas.sql.
  • Restauração: O comando mysql importa o arquivo backup_banco_tarefas.sql de volta para o banco banco_tarefas.

O que é SSH?

O SSH (Secure Shell) permite acessar e transferir arquivos entre uma máquina local e um servidor de forma segura. Exemplos de comandos:

ssh usuario@servidor
scp arquivo.txt usuario@servidor:/caminho/destino

O que é Linux?

O Linux é um sistema operacional open-source amplamente usado em servidores web. Distribuições populares incluem:

  • Ubuntu Server
  • Debian
  • CentOS

Comandos básicos do Linux

  • ls: Lista arquivos e diretórios.
  • cd: Navega entre diretórios.
  • cp: Copia arquivos.
  • rm: Remove arquivos.
  • mkdir: Cria um diretório.
